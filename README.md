# Fluid Mechanics: Pipe Flow Sudden Expansion Analysis

## Overview
This project investigates minor pressure losses in a pipe with a sudden cross-sectional expansion. The study compares traditional theoretical Bernoulli calculations with numerical results from a CFD solver.

## Objectives
* Calculate the minor loss coefficient for a laminar flow.
* Analyze pressure and velocity field transitions at the expansion boundary.
* Evaluate the accuracy of theoretical approximations versus numerical simulations.

## Technical Methodology
* **Theoretical**: Applied Bernoulli and Darcy-Weisbach equations.
* **Numerical**: Conducted 2D steady-state simulations using ANSYS Fluent.
* **Analysis**: Investigated flow development requirements, noting that entry lengths significantly impact result correlation.

## Software Tools
* **ANSYS Fluent**: Numerical simulation and post-processing.
* **Engineering Calculations**: Theoretical derivation and verification.
